## Run locally

After cloning the repository, simply type in the following commands:

for Unix-like systems:

```shell

```

for Windows systems:

```shell
.\run.cmd
```

If you are using [CLion](https://www.jetbrains.com/ru-ru/clion/) just, press the
**run** button and check the result.
